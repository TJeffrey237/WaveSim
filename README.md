# WaveSim
By Tyler Jeffrey

## Overview
This project uses a buoyancy simulation to play a small boat-related game.

Players must collect all pieces of trash from the sea and deposit it in a designated collection area to complete the game.

## Mechanics
Boat Movement:
- WASD (Left/Right/Forward/Back)

Trash Collection/Depositing:
- Collect: Simply touch the trash pieces with the boat.
- Deposit: Navigate into collection zone (indicated by particles). All held trash will be immediately deposited

## Installation & Setup
1.  Clone this repository to your local machine.
2.  Open the project in **Unity 2022.3 LTS** (or newer).
3.  Ensure all scenes are added into **Build Settings**.
4.  Press **Play** in the `MainMenu` scene to begin the experience.
